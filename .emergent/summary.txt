**PRODUCT REQUIREMENTS:**

-   **Future/Optional:**

-   **Technical/Design Specs:**
    -   The UI must have a modern Kiosk Mode look and feel, inspired by self-scan terminals.
    -   The currency is Surinamese Dollar (SRD).
    -   All user-facing text must be in Dutch (Nederlands).
    -   The system must support multiple landlord companies (multi-tenant), where each company can register and manage their own properties and tenants.
    -   Receipts (Kwitantie) must be a modern A4 design, and the printed version must exactly match the on-screen preview.

**User's preferred language**: Nederlands (Dutch)

**what currently exists?**
A full-stack application has been developed with a FastAPI backend and a React frontend. The application features a dual-interface system:
1.  **Kiosk Mode:** A full-screen, touch-friendly interface for tenants to select their apartment, view balances, and make payments. It includes a modern A4-style receipt (Kwitantie) generation screen with an auto-redirect timer.
2.  **Admin Dashboard:** A password-protected area for landlords, styled similarly to the kiosk, where they can manage apartments, tenants, and view payment histories, including saved digital copies of all receipts.

The system supports automatic billing for the subsequent month after a full rent payment is made. All UI and functionality have been built according to user feedback, including multiple design revisions to achieve a specific kiosk aesthetic.
